ASPL Quick View for WooCommerce makes it easier than ever for customers to browse through your woocmmerce products and make it Add-to-cart.

== Description ==

Let your customers quickly preview products and add-to-cart through a Quick view. 

Features:- 

1) Add 'Quick View' button on WooCommerce shop loop items to display product summary using a popup. 
2) Product image with product gallery, title, star_rating, price, quantity and add to cart button will be shown in popup. 
3) “Quick View” also display on mobile View. 
4) Easy to use For add-to-cart 
5) Supports simple product and variable product types. 
6) Admin can change the button text, button background-color and button text-color.
